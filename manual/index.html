Chapter 1: Basics of Instructions

Instructions are made by using 3, 5, or 14 characters in one big clump. Commands are separated by a space, and 14 digit commands are not separated. Examples are 10110+00000000, 409, 209, 11020, and 31050. An example sequence of commands would be 209 210 220 10110+90000000 10150-10000000 11050. All instructions have 1 argument or more. in 209, 09 is an address argument. in 10110+00000000, 01, 10, and +00000000 are all arguments. In 11010, 10 and 10 are arguments. In this case they are addresses.

 

Chapter 2: Information About Addresses and Stores

The stores:

06 Tape reader 6 (not implemented yet)

08 First 7 digits of accumulator (not implemented yet)
09 All digits of accumulator
10-99 Memory Stores

Every storage place is exactly 9 digits, with one being a + or - sign.You cannot write to any tape readers, but you can read from tape readers. More on that in the next section. Memory stores are where your numbers go for storage. When you restart the WITCH-E, those numbers will go away. This is because the number storage units (also called dekatrons) have no power.

 

Chapter 3: Adding and Clearing to Stores

To add a number to a store, type 101 + the number of store that is not a tape reader, then + or - and an 8 digit number. The 8 digit number has to have zeros before it if it wasn't originally there. For example, 10110+00000000 would work, but 10110 2000000 would not work. It would not work because it is not an 8 digit number, it has a space between the command and the values, and the value does not have a + or - sign. The version of this command that would work would be 10110+02000000. This also works with a 3 instead of a 1 but to subtract. Here's an example: 30110+55555555. Clearing the store is required before setting it. To clear a store, type 2 or 4 + the address number of the store + 00. An example is 40900 or 25500.

 

Chapter 4: Arithmetic

1, 2, 3, and 4 as orders can be used for another purpose too. They can be used for arithmetic. To add the stores to add from must have assigned values. The command is 1 or 2 + store 1 + store 2. An example is 11020 or 22020. To subtract, the command is 3 or 4 + store 1 + store 2. Store 2 is subtracted from store 1 and put in store 2. An example is 31050 or 41930.
